## Manegement
http://tyru.hatenablog.com/entry/2017/12/20/035142
```
git submodule add http~somedeveloper/somename.vim
```

Update at once
```
$ git submodule foreach git pull
$ git commit -a -m 'Update all vim plugins'
```
